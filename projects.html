java-3d-renderer

A real-time 3D rendering engine written in Java using the standard 2D graphics libraries. It can read .obj files and has a built in first-person camera.
Website

ProjectATOD

An APCS final project written in Java using libGDX. As a tailgunner, you must shoot enemy planes off your tail before they shoot you! We chose ATOD because it spells DOTA backwards.

bluebot

A Slack-integrated hubot that drastically increases and decreases Option 16's productivity. It can look up match scores and teams on The Blue Alliance, look up robotics-related terms, and recite the entire bee movie script.

MilkyWhite

The Arduino code for the animatronic, remote-controlled cow named Milky White. It features moving eyelids, ears, and a tail. The cow has been displayed in several of Option 16's outreach activities.

PlaneVTruck

A game made during the CodeDay San Diego Hackathon and winner of the CodeDay San Diego "Most Unique" award. It's a multiplayer split-screen video game where you can play as either a plane or a truck/car. As a plane, your goal is to bomb the truck. As a truck, your goal is to jump off one of the ramps and crash into the plane.

cv-2016

A targeting program written in C++ using OpenCV to detect a U-shaped target made out of reflective tape. This was made for the 2016 FRC Competition, FIRST Stronghold.
